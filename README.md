# Сайт с рецептами "Что в холодильнике?"

Сайт с рецептами и фильтрацией по ингредиентам.
Идея проекта: В базе данных лежат рецепты. Пользователь отмечает галочками продукты, которые у него есть (яйца, молоко, мука), а сайт выдает список блюд, которые можно приготовить именно из этого набора.
